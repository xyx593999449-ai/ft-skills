# 存在性核实规则

本目录包含用于验证POI实体是否真实存在的规则。

## 规则清单

### 1. 高可信度官方来源确认规则

**文件**：`official_confirmation.yaml`

**功能**：通过官方渠道（权重1.0）的确认来判断POI存在性

**应用条件**：
- 至少有1个官方来源的证据
- 官方来源明确表示实体存在（status=active）

**输出**：
- result: pass
- confidence: 0.95

### 2. 多源一致确认规则

**文件**：`multi_source_consensus.yaml`

**功能**：通过多个不同来源的一致证据判断存在性

**应用条件**：
- 至少有2个权重≥0.8的证据
- 所有证据的状态一致

**输出**：
- result: pass
- confidence: 0.90

### 3. 证据不足降级规则

**文件**：`insufficient_evidence.yaml`

**功能**：当证据不足时进行降级处理

**应用条件**：
- 有效证据数 < 2个
- 或没有官方来源证据

**输出**：
- result: uncertain
- confidence: 0.5

## 规则应用顺序

1. 检查官方来源确认（优先级1）
2. 检查多源一致性（优先级2）
3. 检查证据充分性（优先级3）
4. 默认不确定（优先级4）
